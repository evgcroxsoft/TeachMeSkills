Дз на понедельник.
Спроектировать и реализовать REST API сервиса для аренды посуточного жилья.
Сущности: Арендодатель, Апартаменты, Пользователь (снимает жилье), Оценка

1. Арендодатель может иметь несколько апартаментов для посуточной сдачи
2. Пользователь может снимать несколько апартаментов (даже одновременно)
3. Реализовать систему оценок с комментариями (от 1 до 5 звёзд). Система оценок может работать в две стороны: пользователь может оценить апарты, арендодатель может оценить пользователя
4. Использовать async/await
5. Сделать бэкграунд таски для отправки эмейлов: после заявки на аренду со стороны пользователя, после подтверждения заявки на аренду со стороны арендодателя. Вместо отправки настоящего эмейла сделать запись в файл, формат произвольный.
*6. Сделать JWT auth с разными уровнями доступа: для пользователя и арендодателя
